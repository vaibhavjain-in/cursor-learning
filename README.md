# Cursor learning

Notes, experiments, and documentation from learning **Cursor**—as an AI-assisted coding tool and as an **IDE** built on familiar editor foundations.

## What lives here

This repository collects practical learnings: workflows, prompts, settings, integrations, and patterns that stick. It is a personal knowledge base, not a product or tutorial series.

## Documents

- [PyCharm → Cursor migration](PyCharmToCursorMigration.md) — shortcuts, extensions, env/debug setup, and a migration checklist.

## Scope

| Area | Examples |
|------|----------|
| **Cursor as a tool** | Chat, Composer, Agent, rules, skills, context and indexing, prompting, team or project conventions |
| **Cursor as an IDE** | Editing, navigation, extensions, terminals, debugging, Git, keyboard-driven workflows, customization |

## How to use this repo

Organize however it helps you—by topic, by date, or by feature. Short, searchable notes often beat long essays when you need to recall something months later.
